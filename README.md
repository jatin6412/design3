# Elm Grove Liquor Website

This is a simple one-page website for a fictional liquor store called "Elm Grove Liquor".

## Features

-   **Visually Appealing Design:** The website has a modern and clean design with a dark theme and gold accents.
-   **Interactive Background:** A particle animation in the background adds a dynamic and engaging touch.
-   **Product Showcase:** The website showcases the store's products with images and descriptions.
-   **Customer Testimonials:** A section for customer testimonials builds trust and credibility.
-   **Contact Form:** A simple contact form allows users to get in touch with the store.

## Technologies Used

-   **HTML:** The structure of the website is built with HTML.
-   **CSS:** The website is styled with CSS, including a responsive design that adapts to different screen sizes.
-   **JavaScript:** The interactive background animation is created with JavaScript.
-   **Google Fonts:** The website uses custom fonts from Google Fonts to enhance the typography.
-   **Unsplash:** The product images are sourced from Unsplash, providing high-quality and visually appealing images.

## How to Use

To view the website, simply open the `index.html` file in a web browser.
